# How this works

## Using locally
1. git clone this repository to your local computer

```bash
>> git clone git@github.com:NieneB/this-way-cartography-map-styles.git
```

2. Serve the index.html from the demo site 

For example use the LiveServer in VSCode. 

3. Now you can see the map in the browser and edit all the styles available in VSCode with live changes. 

## Edit with Maputnik

Open the files in [Maputnik](https://maplibre.org/maputnik/) 
1. go to **Open**
2. Paste the raw url from a style to **Load from URL**. For example: `https://raw.githubusercontent.com/NieneB/this-way-cartography-map-styles/refs/heads/main/styles/style_NDW_demo_v1.json` 
3. Load and view and edit your style!
